# Se connecter à FutureKawa

L'accès à l'application est protégé : seules les personnes disposant d'un compte
peuvent consulter les données de suivi des stocks.

## Se connecter

1. Ouvrez l'application dans votre navigateur. Si vous n'êtes pas connecté·e,
   vous arrivez automatiquement sur la page **Connexion**.
2. Saisissez votre **adresse email** et votre **mot de passe**.
3. Cliquez sur **Se connecter**.

Une fois connecté·e, vous êtes redirigé·e vers l'écran que vous tentiez
d'atteindre (ou l'accueil par défaut).

> **Mot de passe** : au moins 12 caractères, avec une minuscule, une majuscule
> et un chiffre. Le formulaire vous signale en rouge les champs à corriger.

## En cas d'erreur

- **« Email ou mot de passe incorrect. »** : les identifiants ne correspondent à
  aucun compte. Vérifiez votre saisie (attention aux majuscules). Par sécurité,
  le message reste volontairement générique.
- **« Connexion impossible pour le moment. »** : le service est momentanément
  indisponible. Réessayez dans quelques instants ; si le problème persiste,
  contactez l'administrateur.

## Rester connecté·e

Votre session reste active sans avoir à ressaisir vos identifiants, même après
quelques minutes d'inactivité ou un rafraîchissement de la page. Si votre
session expire définitivement, vous êtes automatiquement renvoyé·e vers la page
de connexion.

## Se déconnecter

Cliquez sur l'icône **utilisateur** en haut à droite, puis sur **Se
déconnecter**. Vous êtes immédiatement renvoyé·e vers la page de connexion.
